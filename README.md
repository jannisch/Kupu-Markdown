# Kupu-Markdown

Kupu is a fast and easy to use online markdown-editor.

## Features

* Export as MD, HTML and TeX
* Super fast live preview
* Share functionality (WIP)
* Clean design

## Technologies

* Slim/Sass as templating languages
* Javascript
* Vue.js for the UI
* Ruby for building

## Build

     # bundler install
     $ ruby build-script.rb

Should do the job
